# Navigation Bundle

BladewindUI navigation bundle — installs all navigation components (Tab, Dropmenu, Pagination, Theme Switcher).

## Installation

```bash
composer require mkocansey/bladewind-navigation
```

### Included packages

| Package | Component |
|---|---|
| `mkocansey/bladewind-tab` | Tab Group |
| `mkocansey/bladewind-dropmenu` | Dropdown Menu |
| `mkocansey/bladewind-pagination` | Pagination |
| `mkocansey/bladewind-theme-switcher` | Theme Switcher |

You can also install the full BladewindUI library in one go:

```bash
composer require mkocansey/bladewind
```

## Documentation

Full documentation, live demos, and all available attributes are at **[bladewindui.com](https://bladewindui.com)**.
