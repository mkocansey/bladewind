# Forms Bundle

BladewindUI forms bundle — installs all form components (Input, Select, Checkbox, Radio, Toggle, Datepicker, Timepicker, Colorpicker, Filepicker, Slider, Checkcards, Textarea, Number, Code).

## Installation

```bash
composer require mkocansey/bladewind-forms
```

### Included packages

| Package | Component |
|---|---|
| `mkocansey/bladewind-input` | Input |
| `mkocansey/bladewind-textarea` | Textarea |
| `mkocansey/bladewind-select` | Select |
| `mkocansey/bladewind-checkbox` | Checkbox |
| `mkocansey/bladewind-radio` | Radio Button |
| `mkocansey/bladewind-toggle` | Toggle / Switch |
| `mkocansey/bladewind-datepicker` | Datepicker |
| `mkocansey/bladewind-timepicker` | Timepicker |
| `mkocansey/bladewind-colorpicker` | Color Picker |
| `mkocansey/bladewind-filepicker` | File Picker |
| `mkocansey/bladewind-slider` | Slider |
| `mkocansey/bladewind-checkcards` | Check Cards |
| `mkocansey/bladewind-number` | Number Stepper |
| `mkocansey/bladewind-code` | Verification / OTP Code Input |

You can also install the full BladewindUI library in one go:

```bash
composer require mkocansey/bladewind
```
